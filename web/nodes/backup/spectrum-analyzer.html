ANALYSIS TECHNIQUES
Higher FFT sizes provide better frequency resolution but slower analysis
Use Hann window for general analysis, Blackman for precise measurements
Overlap of 0.5-0.75 provides smoother spectral updates
Exponential averaging smooths fluctuations for stable display
Peak detection identifies dominant frequency components automatically
Use full frequency range for broadband analysis
Band analysis perfect for octave-based acoustic measurements
Phase analysis reveals timing relationships between components
Spectrogram shows how spectrum changes over time
THD measurement quantifies distortion in processed audio
