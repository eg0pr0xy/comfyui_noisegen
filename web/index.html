Ready to Generate Chaos?

New to noise generation? Start with the Noise Generator (Universal) node!

Install via ComfyUI Manager or manual Git clone. All nodes appear under the NoiseGen category.

7 Noise Types

White, Pink, Brown, Blue, Violet, Perlin, and Band-Limited noise generation with professional-grade algorithms and scientific precision.

Chaos Mixing

11 extreme mixing modes including chaos, XOR, modulo, ring modulation, and more. Perfect for Merzbow-style harsh noise walls.

Professional Quality

32-bit float processing, up to 96kHz sample rates, deterministic random generation, and ComfyUI native AUDIO format support.

Scientific Accuracy

Mathematically correct frequency responses, proper stereo processing, and anti-aliasing for research-grade audio testing.

Extreme Processing

Multi-stage distortion, bit crushing (1-16 bits), feedback delays, and asymmetric clipping for maximum audio destruction.

Multiple Formats

Export to WAV, FLAC, or MP3 with configurable quality settings. Perfect for both production and archival purposes.
