Professional noise generation nodes for ComfyUI workflows.

Professional Quality: 32-bit float, up to 96kHz sample rate
Scientific Accuracy: Mathematically correct frequency characteristics

Installation

Clone into your ComfyUI custom_nodes directory:

git clone https://github.com/eg0pr0xy/noisegen.git

Usage

Find the NoiseGen nodes in your ComfyUI node browser under the audio category.
