GENERATOR

Foundation noise generator with 7 scientifically-accurate noise types. Professional controls for amplitude, sample rate, stereo processing, and seed-based reproducibility.
INPUTS
noise_type
ENUM
white, pink, brown, blue, violet, perlin, bandlimited
Noise generation algorithm with distinct frequency characteristics.
duration
FLOAT
0.1 - 300.0 seconds
Length of generated audio.
sample_rate
ENUM
8000, 16000, 22050, 44100, 48000, 96000 Hz
Audio sample rate. Higher rates = better frequency response.
amplitude
FLOAT
0.0 - 2.0
Output amplitude multiplier. 1.0 = unity gain.
seed
INT
0 - 2147483647
Random seed for reproducible generation.
channels
ENUM
1 (mono), 2 (stereo)
Number of audio channels.
stereo_mode
ENUM
independent, correlated, decorrelated
Stereo channel relationship mode.
stereo_width
FLOAT
0.0 - 2.0
Stereo width control. 0.0 = mono, 2.0 = extra wide.
OUTPUTS
audio
AUDIO
32-bit float, specified sample rate
Generated noise audio in ComfyUI AUDIO format.

WHITE NOISE

Equal energy across all frequencies. Perfect for harsh noise, testing, and processing chains.

PINK NOISE

1/f frequency spectrum. Natural-sounding, great for ambient textures and sleep applications.

BROWN NOISE

1/f² spectrum with enhanced low frequencies. Excellent for deep, rumbling textures.

BLUE NOISE

Higher frequencies emphasized. Bright, crisp character ideal for high-frequency content.

VIOLET NOISE

Very high frequency emphasis. Sharp, cutting character for extreme high-end content.

PERLIN NOISE

Smooth, organic variations. Perfect for natural-sounding wind and atmospheric effects.

BANDLIMITED

Precisely filtered frequency range. Essential for scientific measurements and testing.

PRO TIPS
Use white noise with high amplitude (0.8-1.0) as input for extreme processing chains
Pink noise at low amplitudes (0.1-0.3) works perfectly for sleep applications
Brown noise provides excellent bass content for subwoofer testing
Set identical seeds across sessions for reproducible scientific measurements
Use bandlimited mode to test specific frequency ranges in audio equipment
Perlin noise creates organic, wind-like textures perfect for film scoring
Higher sample rates (48kHz, 96kHz) crucial for spectral processors
Independent stereo mode creates maximum chaos for harsh noise applications
Combine multiple generators with different seeds for layered textures
Amplitude values above 1.0 create clipping - sometimes desired for harsh noise
